> You wouldn't import a bear

![a bear](bears/polar.jpg)

This is how you don't import carnivores.

## Install

```
npm i -g a-bear
```

## Usage

How can I access the bears?

```
import bear from "a-bear"

var grizzlyBear = bear("grizzly")
```

### What bears can I access?
- Grizzly
- Polar
- Panda
- Koala
